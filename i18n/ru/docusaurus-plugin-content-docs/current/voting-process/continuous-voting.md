# Непрерывное голосование

## Принцип

**Непрерывное голосование** означает, что гражданин, имеющий законное право голоса, может **в любой момент времени** отдать свой голос по любому вопросу, вынесенному на голосование. Нет фиксированного дня выборов — голосование происходит непрерывно.

Если гражданин ранее уже голосовал за определённый вариант, а теперь поменял мнение, его голос **перемещается** от одного варианта к другому. Таким образом, соотношение голосов меняется постоянно, отражая текущую волю общества.

## Два типа голосования

### Выборное голосование

Цель — определить, кто из претендентов займёт выборную должность.

**Механика:**

1. Граждане могут голосовать за любого кандидата из списка в любой момент
2. Текущее соотношение голосов публикуется ежедневно
3. Когда кандидат набирает больше голосов, чем действующий чиновник, начинается **избирательная гонка**
4. В ходе гонки ежедневно подсчитываются **избирательные очки (ИО)** — разница между процентами голосов
5. Очки накапливаются день за днём
6. Кандидат побеждает при достижении порога (например, 3700 ИО) или проигрывает, если все очки потеряны

### Решающее голосование (по вопросам и законам)

Цель — принять решение по конкретному вопросу или закону.

**Механика:**

1. Гражданин или должностное лицо предлагает вопрос
2. Вопрос проходит техническую проверку
3. Для вынесения на голосование вопрос должен набрать поддержку (например, 10% избирателей)
4. После набора поддержки начинается непрерывное голосование «за» / «против»
5. Когда проголосует достаточная доля электората (например, 60%), запускается накопительный подсчёт
6. Победитель определяется по достижении порога ИО

## Накопительная система очков

Ключевой механизм непрерывного голосования — **накопительный подсчёт избирательных очков (ИО)**.

### Формула

Ежедневный расчёт ИО для выборного голосования:

```
ИО(день) = P(кандидат) - P(чиновник)
```

Где `P(X)` — процент голосов от общего числа избирателей за кандидата X.

Накопленные очки:

```
ИО(накопленные) = Σ ИО(день) за все дни гонки
```

### Пример

Рассмотрим динамику избирательной гонки:

| День | Кандидат (%) | Президент (%) | Разница (ИО) | Накоплено (ИО) |
| ---- | ------------ | ------------- | ------------ | -------------- |
| 1    | 35           | 35            | старт        | 0              |
| 2    | 36           | 34            | +2           | 2              |
| 3    | 37           | 34            | +3           | 5              |
| 4    | 36           | 35            | +1           | 6              |
| 5    | 35           | 35            | 0            | 6              |
| 6    | 33           | 36            | −3           | 3              |

Как видно, преимущество кандидата может как увеличиваться, так и уменьшаться. Гонка продолжается до победы одного из участников.

### Примеры сроков при пороге 3700 ИО

| Преимущество кандидата | Необходимый срок   |
| ---------------------- | ------------------ |
| 10%                    | ~370 дней (~1 год) |
| 25%                    | ~148 дней          |
| 50%                    | ~74 дня            |
| 100%                   | ~37 дней           |

## Публикация результатов

Информация о текущем соотношении голосов публикуется **ежедневно** и доступна всем гражданам. Это обеспечивает:

- **Прозрачность** — каждый видит текущую расстановку сил
- **Предсказуемость** — тренды изменения поддержки видны заблаговременно
- **Подотчётность** — действующие чиновники видят уровень своей поддержки в реальном времени

## Преимущества

1. **Постоянная обратная связь** от общества к власти
2. **Оперативная реакция** на изменения в обществе
3. **Отсутствие «мёртвого периода»** — граждане влияют на власть каждый день
4. **Ликвидация асимметрии** между влиянием власти и народа
