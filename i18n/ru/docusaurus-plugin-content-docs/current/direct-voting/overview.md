# Прямое голосование: обзор

## Определение

**Прямое голосование** — это механизм, при котором каждый гражданин лично и непосредственно участвует в принятии политических решений, без обязательного посредничества представителей. В отличие от представительной демократии, где граждане выбирают депутатов, которые затем голосуют от их имени, прямая демократия предоставляет каждому право голоса по каждому вопросу.

## Зачем нужно прямое голосование

Фундаментальное инвариантное свойство демократии — это **принятие властного решения через голосование**. Демократия — это система:

1. **Определения политической воли народа** посредством принятия решений через голосование
2. **Осуществления принятых решений** выборными чиновниками

Смысл голосования заключается в **разрешении конфликтов с минимальными потерями** для общества путём численной оценки противостоящих сил. Голосование заменяет силовое столкновение сторон более выгодным для общества численным сравнением.

## Отличия предлагаемой системы

Предлагаемая система прямого голосования отличается от существующих моделей по нескольким ключевым параметрам:

### По сравнению с традиционной представительной демократией

| Параметр            | Традиционная                     | Предлагаемая          |
| ------------------- | -------------------------------- | --------------------- |
| Частота голосования | Раз в 4–7 лет                    | Непрерывно            |
| Предмет голосования | Выбор представителя              | Каждый закон и вопрос |
| Верификация         | Доверие комиссиям                | Криптографическая     |
| Вес голоса          | Равный                           | С учётом компетенций  |
| Делегирование       | Нет (только через представителя) | Да, с правом override |

### По сравнению с прямой демократией (например, Швейцария)

| Параметр                  | Швейцарская модель                  | Предлагаемая              |
| ------------------------- | ----------------------------------- | ------------------------- |
| Инициирование референдума | Сбор подписей (сложная процедура)   | Через систему голосования |
| Периодичность             | По мере инициирования               | Непрерывно                |
| Технология                | Бюллетени / электронное голосование | Блокчейн + GPG            |
| Необходимость одобрения   | Да                                  | Нет                       |
| Учёт компетенций          | Нет                                 | Да                        |

## Ключевые компоненты системы

Система прямого голосования включает в себя несколько взаимосвязанных механизмов:

- **[Непрерывное голосование](../voting-process/continuous-voting.md)** — возможность голосовать в любой момент
- **[Инерционность](../voting-process/inertia.md)** — накопительный механизм, защищающий от импульсивных решений
- **[Делегирование](../voting-process/delegation.md)** — передача голоса доверенному лицу
- **[Блокчейн и криптография](../voting-process/blockchain.md)** — технологическая основа
- **[Весовые коэффициенты](../competency-weights/overview.md)** — учёт профессиональных компетенций
