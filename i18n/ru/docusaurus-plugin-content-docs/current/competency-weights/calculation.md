# Расчёт весов

## Основная формула

Вес голоса гражданина при голосовании по конкретному вопросу определяется как:

```
Вес_голоса = 1 + Σ (Доля_компетенции_i × Множитель_i)
```

Где:

- `1` — базовый вес голоса (есть у каждого гражданина)
- `Доля_компетенции_i` — процент i-й компетенции в распределении данного вопроса
- `Множитель_i` — усиление за наличие i-й компетенции

Для граждан **без профессиональных компетенций** вес голоса равен **1** (базовый).

## Развёрнутый пример

### Условия

Рассмотрим голосование по **закону о запрете ГМО**.

**Распределение компетенций:**

| Компетенция                   | Доля |
| ----------------------------- | ---- |
| Генетика                      | 70%  |
| Экология                      | 15%  |
| Экономика сельского хозяйства | 10%  |
| Общественное здоровье         | 5%   |

**Участники голосования:**

- Общее число проголосовавших: **100 000** человек
- Из них профессиональных генетиков: **500** человек
- Остальные компетенции для упрощения не учитываем

### Расчёт

**Шаг 1: Определение множителя для генетиков**

Множитель для компетенции определяется через долю этой компетенции:

```
Множитель_генетики = Доля_генетики × 100 = 0.70 × 100 = 70
```

Коэффициент 100 — базовый коэффициент усиления, который может настраиваться в зависимости от уровня голосования.

**Шаг 2: Расчёт весов голосов**

| Когорта          | Кол-во      | Вес голоса  | Суммарный вес     |
| ---------------- | ----------- | ----------- | ----------------- |
| Обычные граждане | 99 500      | 1           | 99 500            |
| Генетики         | 500         | 1 + 70 = 71 | 500 × 71 = 35 500 |
| **Итого**        | **100 000** |             | **135 000**       |

**Шаг 3: Анализ**

- Весовой вклад 500 генетиков: `35 500 / 135 000 ≈ 26.3%` итогового результата
- Весовой вклад 99 500 остальных: `99 500 / 135 000 ≈ 73.7%` итогового результата

Таким образом, **0.5% участников** (генетики) определяют **26.3%** итогового результата, что отражает их профессиональную значимость для данного вопроса.

## Пример с несколькими компетенциями

Рассмотрим гражданина, который имеет компетенции в генетике **и** экологии.

```
Вес_голоса = 1 + (0.70 × 100) + (0.15 × 100) = 1 + 70 + 15 = 86
```

Его голос весит 86 — значительно больше, чем у специалиста только в одной области.

## Пороговые значения

### Минимальный порог компетенции

Не каждый обладатель диплома автоматически получает полный множитель. Множитель может масштабироваться в зависимости от **уровня компетенции** (определяемого peer-based системой):

| Уровень компетенции                   | Процент от полного множителя |
| ------------------------------------- | ---------------------------- |
| Начальный (только диплом)             | 25%                          |
| Средний (диплом + peer-оценка)        | 50%                          |
| Высокий (значительная peer-оценка)    | 75%                          |
| Экспертный (максимальная peer-оценка) | 100%                         |

**Пример для генетика:**

- Начальный уровень: `1 + 70 × 0.25 = 18.5`
- Экспертный уровень: `1 + 70 × 1.0 = 71`

### Максимальный вес

Для предотвращения чрезмерного влияния одного человека может быть установлен **потолок** максимального веса голоса:

```
Вес_голоса = min(Рассчитанный_вес, Макс_вес)
```

Рекомендуемый потолок: **100** (голос не может весить более чем в 100 раз больше базового).

## Edge cases

### Вопрос без компетенций

Если вопрос не отмечен никакими компетенциями (например, вопрос общественной морали), **все голоса имеют одинаковый вес 1**. Система весов не применяется.

### Очень мало профессионалов

Если по данной компетенции в системе зарегистрировано очень мало профессионалов (например, 5 человек), их совокупный вес может оказаться непропорционально большим. В таких случаях может применяться **корректирующий коэффициент**.

### Конфликт интересов

Профессионалы, имеющие прямой финансовый или иной интерес в исходе голосования, могут быть **отмечены** в системе (на основании публичных деклараций). Это не лишает их права голоса, но информация о возможном конфликте интересов доступна всем.

## Итоговая таблица формул

| Параметр              | Формула                                           |
| --------------------- | ------------------------------------------------- |
| Базовый вес           | `1`                                               |
| Множитель компетенции | `Доля_компетенции × Базовый_коэффициент_усиления` |
| Вес с учётом уровня   | `Множитель × Процент_уровня_компетенции`          |
| Итоговый вес голоса   | `1 + Σ(Вес_с_учётом_уровня_i)`                    |
| Ограниченный вес      | `min(Итоговый_вес, Макс_вес)`                     |
| Суммарный вес когорты | `Кол-во_людей × Вес_голоса`                       |
