# Система весов компетенций

## Проблема равного веса голосов

В традиционной демократии все голоса имеют одинаковый вес: голос профессора генетики и голос домохозяйки при голосовании по закону о ГМО считаются равноценными. Это приводит к ситуации, когда **решения по специализированным вопросам определяются мнением большинства, не обладающего необходимыми знаниями**.

Это не означает, что мнение неспециалистов не важно — оно учитывается. Но профессионалы, обладающие глубокими знаниями в предмете голосования, должны иметь **больший вес** при принятии решений в своей области.

## Принцип весовых коэффициентов

Каждый закон или вопрос, выносимый на голосование, может быть **отмечен набором компетенций**, необходимых для профессионального принятия решения о его целесообразности.

### Распределение компетенций

Все заявленные для закона компетенции составляют **100%**. Каждой компетенции назначается процент, отражающий её значимость для данного вопроса.

**Пример: закон о запрете ГМО**

| Компетенция                   | Доля     |
| ----------------------------- | -------- |
| Генетика                      | 70%      |
| Экология                      | 15%      |
| Экономика сельского хозяйства | 10%      |
| Общественное здоровье         | 5%       |
| **Итого**                     | **100%** |

### Модификатор веса

Граждане, обладающие **профессиональным уровнем требуемых компетенций**, получают модификатор веса голоса в соответствии с распределением компетенций.

Модификатор рассчитывается как:

```
Модификатор = Доля_компетенции × Коэффициент_усиления
```

Где `Коэффициент_усиления` определяется соотношением между количеством профессионалов и общим числом голосующих (подробнее в разделе [Расчёт весов](./calculation.md)).

## Зачем это нужно

### Повышение качества решений

Когда профессионалы имеют больший вес при голосовании по вопросам своей компетенции, решения становятся более обоснованными и взвешенными.

### Сохранение демократии

При этом голоса **всех граждан учитываются**. Непрофессионалы не лишаются права голоса — их голоса просто имеют базовый вес (1.0). Это не элитаризм, а **механизм повышения компетентности** решений.

### Защита от популизма

Популистские решения, противоречащие научному консенсусу, имеют меньше шансов быть принятыми, потому что профессионалы могут компенсировать своим повышенным весом голоса тех, кто поддался манипуляции.

### Мотивация к образованию

Система создаёт естественную мотивацию для граждан получать образование и развивать компетенции, так как это увеличивает их влияние при голосовании.

## Рекомендации экспертов

Помимо взвешенного голосования, эксперты могут публиковать **публичные рекомендации** по вопросам в своей компетенции. Это создаёт информационный слой, который помогает всем гражданам принимать осознанные решения, не подменяя их голос.

### Как работают рекомендации

1. **Публикация рекомендации:** эксперт публикует свою позицию (За / Против / Воздержался) с кратким обоснованием, подписанным криптографически
2. **Агрегация:** рекомендации объединяются по компетентностным группам — например, «Генетики: 87% из 500 рекомендуют ПРОТИВ. Средняя репутация ответивших: 0.82»
3. **Отображение:** рекомендации показываются рядом с вопросом голосования и видны гражданам до того, как они сделают выбор
4. **Опциональность:** эксперт не обязан публиковать рекомендацию. Отсутствие сигнала — тоже информация: сообщество экспертов может не иметь консенсуса, или вопрос лежит вне их компетенции

### Назначение

- Эксперты **информируют**, а не правят — каждый голос гражданина сохраняет базовый вес
- Рекомендации служат ориентиром, особенно для сложных специализированных вопросов
- Прозрачность: позиция и обоснование каждого эксперта публично проверяемы

## Определение компетенций

Кто определяет, какие компетенции необходимы для голосования по конкретному вопросу?

Распределение компетенций определяется **автоматически** через краудсорсинговый механизм:

1. **Заявление релевантности:** при открытии вопроса для голосования зарегистрированные профессионалы могут заявить, что вопрос относится к их области компетенции
2. **Автоматический расчёт:** распределение компетенций вычисляется пропорционально количеству экспертов, заявивших релевантность в каждой области
   - Пример: если 40 генетиков и 60 экологов заявили релевантность, распределение — 40% генетика, 60% экология
3. **Защита от спама:** эксперты, заявляющие релевантность к явно нерелевантным вопросам, теряют репутацию. Сообщество может оспаривать заявления о релевантности через голосование
4. **Запасной вариант:** если ни один эксперт не заявил релевантность к вопросу, система весов не применяется — все голоса имеют равный вес

Подробнее о том, как определяется, кто является профессионалом, читайте в разделе [Назначение компетенций](./assignment.md).
